# ConfigPilot v1.0.20

本版本将内置界面引擎升级至 PrismQML 0.3.1.27，优化按钮下拉菜单在远程桌面环境中的窗口与焦点交互。

## 主要更新

- PrismQML 从 `0.3.1.4` 升级至 `0.3.1.27`。
- 按钮下拉菜单改为在应用主窗口内渲染，不再为这类菜单创建第二个原生窗口。
- 移除按钮下拉打开时的额外原生窗口 `raise` 与焦点激活路径，降低远程桌面下原生 surface 和焦点切换造成界面阻塞的风险。
- 保留其他需要跨窗口显示的菜单、选择器和弹层的原有行为，避免扩大兼容性影响。

## 升级说明

ConfigPilot 已包含应用内更新器。Windows 用户可在应用内完成下载、静默安装和重启；macOS 用户检测到新版后会打开官方 Release 页面。

本版本更新了内置 PrismQML 引擎，必须安装 `v1.0.20` 的新程序包；仅更新配置文件不会获得新的下拉菜单实现。

## 下载

- Windows：`ConfigPilot_Setup_1.0.20.exe`
- macOS Apple Silicon：`ConfigPilot_1.0.20_arm64.dmg`

## macOS 首次安装须知

当前 macOS 版本仅支持 Apple Silicon（M 系列芯片），不支持 Intel Mac。

维护者目前没有 Apple Developer Program 会员，因此 DMG 尚未使用 Developer ID 签名和 Apple 公证。macOS Gatekeeper 可能显示“ConfigPilot 已损坏，无法打开”，但这不代表 DMG 文件真的损坏。

请先把 `ConfigPilot` 拖入 `Applications`，再打开“终端”执行：

```bash
xattr -dr com.apple.quarantine "/Applications/ConfigPilot.app"
open "/Applications/ConfigPilot.app"
```
